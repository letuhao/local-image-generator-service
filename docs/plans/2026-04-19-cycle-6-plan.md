### Chunk 5 — Civitai fetcher (HTTP + streaming + SHA)

Files:
- `app/loras/civitai.py` (new) — `CivitaiFetcher` class
  - `__init__(store, loras_root, registry, api_token, http_client, dir_max_bytes, file_max_bytes, recent_use_days, max_concurrent=1)`
  - `_fetch_metadata(version_id)` — GET `https://civitai.com/api/v1/model-versions/{vid}` with Bearer, tenacity 3x on 5xx
  - `_stream_download(download_url, tmp_path, expected_size)` — httpx streaming w/ per-chunk 30s read timeout, overall 1800s asyncio.timeout, 1 MiB chunks, SHA256 streaming hash, progress update every 4 MiB
  - `_write_sidecar(sidecar_path, meta)` — atomic tmp+rename
  - `enqueue(request_id)` — spawn bg task under semaphore + per-version Lock
  - `close()` — cancel in-flight
  - Private per-version lock dict (weak-refs; cleaned up when last holder releases)
- `tests/test_civitai_fetch.py` (new, 10 tests; respx for HTTP mocking)

Verify: `uv run pytest tests/test_civitai_fetch.py -q` green.
